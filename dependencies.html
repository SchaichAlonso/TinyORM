Dependencies

The code was developed on MSVC 16.9-16.11.16, MSVC 17.2, GCC 10.2-12.1, and Clang 11-14, so may be assumed it will work on future releases of these compilers. Minimum required ISO C++ standard is c++20. The Qt framework version used during development was 5.15.2 and >= 6.2.

Requiredâ€‹
  • minimum ISO C++ standard is c++20
  • >= Qt Framework 5.15.2 - QtCore and QtSql modules (MSVC 2019 binaries)
  • >= Qt Framework 6.2 - QtCore and QtSql modules (MSVC 2019 binaries)
  • >= range-v3 0.11.0
  • tabulate
Optionalâ€‹
  • MySQL Connector/C 8 - used only for the mysql_ping function and provided by MySQL 8 Server

Install dependenciesâ€‹

On Linux, you can install dependencies with the package manager.

MySQL C library
Arch   - pacman -S mariadb-libs
Gentoo - emerge dev-db/mysql (package.use: -server -perl)
Ubuntu - apt install libmysqlclient-dev
range-v3 library (header only)
Arch   - pacman -S range-v3
Gentoo - emerge dev-cpp/range-v3
Ubuntu - apt install librange-v3-dev
ccache
Arch   - pacman -S ccache
Gentoo - emerge dev-util/ccache
Ubuntu - apt install ccache
